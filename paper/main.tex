\begin{abstract}
Every piece of structured knowledge is a finite string over a finite alphabet. We define \emph{semantic simplicial complexes}---simplices that carry hash identifiers, key-value content, and embedding coordinates. The full machinery of algebraic topology runs on semantic simplicial complexes, and every topological operation acquires a precise knowledge-management interpretation. By introducing LLM embeddings as a semantic metric, topology and geometry coexist; their inconsistency pinpoints structural gaps in a knowledge base. We prove an \emph{embedding obstruction theorem}: the topological complexity of a knowledge base gives a lower bound on embedding dimension, and give formal characterizations of hallucination, scaling saturation, ``understanding vs.\ memorization,'' and chain-of-thought reasoning. The algebraic core is mechanically verified in Lean~4 (zero \texttt{sorry}). A reference implementation is an open-source desktop application whose plugin architecture directly reflects the categorical structure.
\end{abstract}
