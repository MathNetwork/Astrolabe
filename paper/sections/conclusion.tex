The capacity formula (Proposition~\ref{prop:profile-capacity}) gives a recursive generating-function formula for the combinatorial size of each refined stage. These yield profile-level invariants: complexity $\mathfrak{C}_{\boldsymbol{\mu}}$, density $\rho_{\boldsymbol{\mu}}$, and spectral entropy $S^{(j)}_{\boldsymbol{\mu}}$.
\item Discrete Hodge decomposition on weighted Astrolebes.
\item Topological cryptography via $h_{\mathrm{id}}$.
\item Simplicial attention mechanisms for Transformer architectures.

\item Core structure time series and structural health monitoring.
\end{itemize}
