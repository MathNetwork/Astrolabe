\subsection{Providing a Networked Annotation Framework for AI Formalization}
\label{sec:disc-ai-annotation}

% Current AI4Math data annotation is point-wise: individual statement/proof pairs
% Astrolabe makes annotation networked: creating obj/mor simultaneously annotates three layers:
%   (1) content and type, (2) relationship types, (3) structural position (auto-computed by analysis plugins)
% One mor creation annotates relationship between two obj --- higher density than pair-wise annotation
% AI training significance: AI sees typed dependency network, not isolated pairs
% Network analysis signals are automatic annotation --- centrality, community, critical path positions
% Paradigm shift: from "manual annotation per item" to "user builds knowledge network + algorithm extracts macro signals"
