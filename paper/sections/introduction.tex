\subsection{The Problem: Second-Class Citizenship of Relations}
\label{sec:intro-problem}

% - Almost all KM systems (Obsidian, Notion, Logseq, wikis) treat relations as node appendages
% - Links are just lines from A to B with no identity, type, or content
% - Not marginal: in any structured knowledge domain (math, law, biology, software architecture), relations carry critical information
% - "Theorem A implies Theorem B" vs "Theorem A uses Theorem B" are completely different semantics, rendered as the same unlabeled line
% - Deeper: without independent identity, relations cannot be referenced, searched, or analyzed
% - Network analysis algorithms need typed edges to produce meaningful signals
