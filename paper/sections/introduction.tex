\section{Introduction}
\label{sec:introduction}

\subsection{The Problem}

Structured knowledge is scattered across heterogeneous representations. The core limitation of existing tools lies in morphisms lacking open semantics. Knowledge graph edges are pre-defined enumerations. Obsidian/Logseq links have no type. Ologs lack a first-principles foundation.

\subsection{Core Insight}

Every externalized form of human knowledge is a finite string over a finite alphabet. Different parsing procedures extract different structures from the same string.

\subsection{Contributions}

\begin{enumerate}
\item From finite alphabets, we define semantic simplicial complexes.
\item Embedding obstruction theorem: embedding dimension lower bound $d \geq k^*$.
\item Formal characterizations of hallucination, scaling saturation, understanding vs.\ memorization, and chain-of-thought.
\item Long exact sequences and the snake lemma applied to version upgrades and modular refactoring.
\item Lean~4 mechanized verification (algebraic core) + experimental validation (semantic metric).
\item Open-source desktop application where plugins = functors.
\end{enumerate}
