\section{Related Work}
\label{sec:related}

\begin{itemize}
\item \textbf{Ologs} (Spivak \& Kent, 2012): No $A^*$ foundation, no simplicial complexes, no metric, no verification.
\item \textbf{AlgebraicJulia / Catlab.jl}: No verification, no knowledge management application.
\item \textbf{TDA for LLM} (HalluZig, TOHA, G-Detector): Empirical approaches that construct complexes from data. This work defines complexes from knowledge structure and explains \emph{why} TDA is effective.
\item \textbf{RDF/OWL}: Edge types are fixed enumerations.
\item \textbf{Obsidian / Logseq}: Edges have no semantics.
\item \textbf{leanblueprint}: Static, manual correspondence between blueprint and formalization.
\end{itemize}
