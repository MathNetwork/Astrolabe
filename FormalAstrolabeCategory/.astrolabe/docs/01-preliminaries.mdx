# §1 Preliminaries

Standard category theory. Not our contribution.

- **Category**: objects, morphisms, composition, identity. Associativity and unit laws.
- **Functor**: structure-preserving map between categories. Preserves identity and composition.
- **Natural transformation**: coherent family of morphisms between functors. Naturality square commutes.
- **Signature** (quiver): $(V, E, s, t)$. Determines generators of a category.
- **Free category**: $\text{Free}(\Gamma)$. Objects = $V$, morphisms = composable paths of edges. Composition = path concatenation.
- **Faithful functor**: injective on morphisms.
- **Enriched category**: Hom-sets enriched over a monoidal category.
- **Quotient category**: $\mathcal{C}/{\sim}$ where $\sim$ is compatible with morphism structure.
- **Initial object**: unique morphism to every other object.
- **Universe hierarchy**: $\text{Type } 0 : \text{Type } 1 : \text{Type } 2 : \cdots$
