# §2 Astrolabe Category

All new definitions, propositions, theorems. Examples from reference implementation interleaved.

## 2.1 Astrolabe Signature

**Definition (Info record).** $\text{Info} := \text{String} \to \text{Option}(\text{Value})$. Open key-value structure.

**Definition (Astrolabe signature).** A triple $\Sigma = (O, M, h)$ where:
- $O : \text{Type } u$, equipped with $\text{info}_O : O \to \text{Info}$
- $M : \text{Type } u$, equipped with $s, t : M \to O$ and $\text{info}_M : M \to \text{Info}$
- $h : O + M \to \text{Hex}_{12}$, injective

Objects and morphisms are first-class citizens sharing identifier space $\text{Hex}_{12}$.

**Definition (Metadata space).** $\mathcal{M}$ is a set of designated keys in Info. $\mathcal{M} \subset \mathcal{M}'$ means $\mathcal{M}'$ has more keys.

**Definition (Signature morphism).** A pair $\phi = (\phi_O, \phi_M)$ mapping objects and morphisms between signatures while preserving source and target:

$$
s(\phi_M(e)) = \phi_O(s(e)), \quad t(\phi_M(e)) = \phi_O(t(e))
$$

Signature morphisms compose: $(\psi \circ \phi)_O = \psi_O \circ \phi_O$, $(\psi \circ \phi)_M = \psi_M \circ \phi_M$.

**Example (signature.json).** In the reference implementation, $O$ is the `"obj"` dictionary (key = $h$, value = open info), $M$ is the `"mor"` dictionary (key = $h$, value includes `source`, `target`, and open info). The storage is field-agnostic — it only requires $h$ for obj, and $h + s + t$ for mor. All other keys are written by functors.

## 2.2 Astrolabe Category

**Definition.** $\mathcal{A}(\Sigma) := \text{Free}(O, M, s, t)$. The free category generated by signature $\Sigma$.

**Proposition (Well-definedness).** $\mathcal{A}(\Sigma)$ satisfies the four category axioms:
1. Identity: empty path $\epsilon_o$ at each object
2. Composition closure: path concatenation
3. Associativity: list concatenation is associative
4. Unit laws: empty list concatenation

**Proposition (Representability).** $\mathcal{A}(\Sigma)$ is completely determined by its generators $(O, M, h)$. Composite morphisms are computed on demand via the universal property of free categories.

**Theorem (Universal property).** For any category $\mathcal{C}$ and quiver morphism $\phi : \Sigma \to \mathcal{C}$, there exists a unique functor $F : \mathcal{A}(\Sigma) \to \mathcal{C}$ with $F \circ \text{generators} = \phi$.

## 2.3 Quotient Category

**Definition (Equivalence from morphisms).** Morphisms with info.sort = "equivalent" induce an equivalence relation $\sim$ on objects: symmetric (automatic reverse mor), transitive (composition).

**Proposition (Quotient well-definedness).** $\mathcal{A}(\Sigma)/{\sim}$ is a well-defined quotient category. The relation $\sim$ is compatible with morphism structure: given $f : a \to b$, $a \sim a'$, $b \sim b'$, construct $f' := p_b \circ f \circ p_a^{-1}$.

**Proposition.** The projection $\pi : \mathcal{A}(\Sigma) \to \mathcal{A}(\Sigma)/{\sim}$ preserves composition and identity.

**Example (Content addressing).** Two signatures merged. Objects with same content hash $h_c$ get equivalent mor. The induced $\sim$ satisfies reflexivity, symmetry, transitivity, morphism compatibility.

## 2.4 Astrolabe Universe

**Definition.** $\mathcal{U} : \text{Type}(u+1)$ such that for any $\mathcal{A}(\Sigma) : \text{Type } u$, there exists a faithful functor $\iota_\Sigma : \mathcal{A}(\Sigma) \hookrightarrow \mathcal{U}$.

**Proposition (Enrichment closure).** For $\mathcal{M} \subset \mathcal{M}'$, the diagram commutes: $\iota = \iota' \circ j$ where $j$ is the inclusion.

**Proposition (Quotient closure).** For compatible $\sim$, the diagram commutes: $\iota = \iota'' \circ \pi$.

**Corollary (Content addressing compatibility).** $h_c(o_1) = h_c(o_2) \implies \iota_{\Sigma_1}(o_1) = \iota_{\Sigma_2}(o_2)$ in $\mathcal{U}$.

## 2.5 Functors on $\mathcal{A}(\Sigma)$

**Theorem (Import functor).** Any signature morphism $\phi$ preserving source and target extends to a functor $F_\phi : \text{Free}(\mathcal{E}) \to \mathcal{A}(\Sigma)$.

**Theorem (Import functor preserves composition).** $F_{\psi \circ \phi} = F_\phi \circ F_\psi$.

**Theorem (Injective import is faithful).** If $\phi$ is injective, $F_\phi$ is faithful on generators.

**Definition (Natural transformation override).** Given $R_1, R_2 : \mathcal{A}(\Sigma) \to \mathcal{V}$ and predicate $\varphi$ on info, define $\eta_A := g_A$ if $\varphi(\text{info}(A))$, else $\eta_A := \text{id}$.

**Proposition (Natural transformation well-definedness).** $\eta$ is a natural transformation: the override decision depends only on info of each object, not on morphisms, so naturality is automatic.

## 2.6 Signature Category

**Definition.** Signature morphisms compose associatively with identity. Signatures and their morphisms form a category $\text{Sig}$.

**Theorem.** $\text{Free} : \text{Sig} \to \text{Cat}$ is a functor: preserves identity ($F_{\text{id}} = \text{id}$) and composition ($F_{\psi \circ \phi} = F_\phi \circ F_\psi$).
