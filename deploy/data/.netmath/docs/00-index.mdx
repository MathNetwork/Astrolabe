# Notes on Geometric Measure Theory

Open-source notes on geometric measure theory, organized as a knowledge graph. The central goal is the **Pitts min-max theorem** <objref id="f57660a4ad6a">[Pit81]</objref>: every compact Riemannian manifold contains a smooth embedded minimal hypersurface (in dimensions $2 \leq k \leq 5$). The notes build up the geometric measure theory foundations needed to state and prove this result.

## Logical Structure

The notes have two parts, linked by the concept of **almost minimizing**:

- **Part I — Existence** (Chapters 1, 3, 4): A min-max variational calculus producing stationary integral varifolds with the *almost minimizing* property on arbitrary compact Riemannian manifolds, in all dimensions and codimensions.

- **Part II — Regularity** (Chapters 5, 6, 7): Curvature estimates for stable hypersurfaces, a decomposition theorem, and regularity theory showing that almost minimizing varifolds are smooth embedded hypersurfaces when $2 \leq k \leq 5$.

Chapter 2 provides the geometric measure theory foundations — currents, varifolds, first variation, regularity — used throughout both parts.
