# NetMath 产品规划

## 域名

mathnetwork.network

## 产品结构

### 官网（mathnetwork.network）

静态网站，介绍 NetMath 是什么：
- 产品介绍、截图、特性说明
- GMTNet 案例展示（截图/视频，不是在线应用）
- 桌面版下载链接
- GitHub 链接

**技术**：纯静态页面，部署到 Vercel 或 GitHub Pages

### 桌面版（NetMath App）

本地 Tauri 应用，官网提供下载：
- 本地创建和编辑知识图谱
- 本地文件，数据在自己电脑上
- 先打磨好再开源

## Git 分支策略

| 分支 | 用途 |
|------|------|
| `main` | 桌面版开发 |
| `legacy` | 旧架构备份 |

## 优先级

1. ~~架构重构~~ ✅ 已完成

3. 桌面版打包 + 发布下载

## 目标用户

- 数学研究者（组织论文笔记、定理依赖关系）
- 数学学生（学习复杂理论的结构）
- 任何需要结构化知识图谱的人
